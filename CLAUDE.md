# Grind2Greatness

## Project Overview
This is a Next.js 16 application using React 19, TypeScript, and Tailwind CSS 4.

## Repository
- **GitHub**: https://github.com/kaedinhanohano/grind2greatness.git
- **Branch**: master
- **Vercel**: Connected to this repository for automatic deployments

**Important**: Always push changes to this repository. This is the main repo for all development.

## Tech Stack
- **Framework**: Next.js 16.1.1
- **UI**: React 19.2.3
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript 5

## Commands
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
